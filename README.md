# MINT PHP Programmer Task 2

## Requirements:
- composer
- symfony(wget https://get.symfony.com/cli/installer -O - | bash)
- mysql
- php 7.2.5 or higher
- apache2


## Build
__Go to mint_php_task2 directory and run in bash.__ <br />
```composer init```

## Run app:
__Go to mint_php_task2 directory and run in bash.__ <br />
```symfony server:start```
